# ESP32-S3 麦克风回放测试

这个项目已经精简成一个单用途固件，用来快速判断麦克风和扬声器链路是否正常。

## 功能

- 按下测试按键后录音 5 秒
- 录音结束后立即通过扬声器回放
- 串口输出录音峰值、平均幅度和是否削波，方便判断麦克风是否几乎无信号

## 当前接线

- 测试按键：`GPIO2`，使用 `INPUT_PULLUP`，按下时接地
- INMP441：
  - `WS -> GPIO5`
  - `SCK -> GPIO4`
  - `SD -> GPIO6`
- MAX98357A：
  - `LRC -> GPIO16`
  - `BCLK -> GPIO15`
  - `DIN -> GPIO7`
- 状态灯：`GPIO21`

如果你的板子接线不同，直接修改 `src/config.h` 即可。

## 使用方式

1. 在 PlatformIO 中编译并烧录
2. 打开串口监视器，波特率 `115200`
3. 按下测试按键
4. 等待录音 5 秒结束
5. 听扬声器回放，同时观察串口日志

## 文件结构

- `src/main.cpp`：按键触发与测试流程
- `src/audio_test.cpp`：I2S 录音与回放
- `src/audio_test.h`：音频接口声明
- `src/config.h`：引脚和录音参数
